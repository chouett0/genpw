# pwgen
## What's it
password generator

<br />

## Require
- python3 > 3.7.4
- string
- secrets
- argparse

<br />

## Usage
```
password: usage: pwgen.py [-h] [-l LENGTH] [-n] [-s]

optional arguments:
  -h, --help            show this help message and exit
  -l LENGTH, --length LENGTH
                        Password Length
  -n, --number          Include Numbers
  -s, --special         Include Special Words
```
